# OOP Modeling and Abstraction

1. `basic_oop.py` - Fundamental OOP concepts
2. `animal_polymorphism.py` - Abstract classes and polymorphism

## How to Run

```bash

python basic_oop.py

# Polymorphism demo
python animal_polymorphism.py

python advanced_oop.py
```
